# 🚀 Goal Manager (NLW Pocket)

A simple CLI application built with Node.js to manage personal goals, focusing on file handling and asynchronous operations.

## 📖 About

This project was developed during Rocketseat's NLW Pocket event.  
It is a command-line application that allows users to manage their goals in a simple and interactive way.

The application provides features to create, list, update, and remove goals, using file storage to persist data.

## 🛠️ Technologies

- Node.js
- JavaScript
- Inquirer
- File System (fs)

## ⚙️ Features

- Create new goals  
- List all goals (open and completed)  
- Mark/unmark goals as completed  
- Remove goals  
- Interactive CLI feedback  

## ▶️ How to run the project

```bash
# Clone the repository
git clone https://github.com/seu-usuario/nlw-pocket-javascript.git

# Navigate to the project folder
cd nlw-pocket-javascript

# Install dependencies
npm install

# Run the application
node index.js
